## $(date +%Y-%m-%d) - Memoization for List Items During Map Traversal
**Learning:** In React components that iterate over lists, performing heavy string/regex processing (like `getFirstImage` and `getExcerpt`) inside the `.map()` loop during render blocks the main thread and can cause O(N) operations per render. This happens even if the list contents themselves haven't fundamentally changed, due to parent component re-renders.
**Action:** When working with Dynamic Post Grids or large item lists, always wrap the data processing logic (e.g., regex, complex filtering) in a `useMemo` that pre-computes the required attributes into a newly mapped array. Do this *before* passing it to the JSX `.map()` function, saving large amounts of processing time on every scroll and state change.
