## 2024-06-25 - Native lazy loading optimization & React Rules of Hooks
**Learning:** Adding `loading="lazy"` to standard `<img>` tags below the fold is a critical performance win, especially in Next.js applications where `<Image />` is restricted due to external domains. However, while making performance improvements, I discovered that existing code contained a severe React Rules of Hooks violation (`useMemo` was placed after an early `return` for `if (loading)`). Fixing these structural issues is necessary before optimization, otherwise the component will fatally crash upon state resolution.
**Action:** Always ensure React components follow the Rules of Hooks (all hooks must be at the top level, before any conditional logic or early returns). When optimizing rendering, concurrently fix underlying lifecycle bugs to prevent breaking changes.
